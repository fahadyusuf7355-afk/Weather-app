# 🌤️ SkyCast

A modern weather application built using HTML, CSS, and JavaScript that provides real-time weather information for any city using the OpenWeather API.

## 🚀 Features

- 🔍 Search weather by city name
- 📍 Get weather using your current location
- 🌡️ Real-time temperature and "Feels Like" data
- 💧 Humidity information
- 🌬️ Wind speed
- ☁️ Dynamic weather icons
- 🖼️ Weather-based background images
- ⌨️ Press Enter to search
- 📱 Responsive glassmorphism UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
